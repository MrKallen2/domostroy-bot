# Telegram-бот «Богдан» для автоматизации строительной компании «ДОМОСТРОЙ»

Программное обеспечение предназначено для расчета параметров фундаментов на винтовых сваях, ведения учета заявок и интеграции с платежной экосистемой.

## Структура проекта
* `bot.py` — основной исполняемый файл запуска бота.
* `config.py` — модуль конфигурации и инициализации переменных окружения.
* `sheets.py` — интеграция с API Google Таблиц (gspread).
* `utils.py` — вспомогательные расчетные функции и хендлеры.

## Инструкция по развертыванию
1. Склонируйте репозиторий.
2. Создайте локальный файл `.env` на основе шаблона `.env.example` и укажите свои API-ключи.
3. Разместите ваш приватный ключ авторизации Google Cloud в файл `google-key.json` по примеру из `google-key.json.example`.
4. Установите зависимости: `pip install -r requirements.txt`
5. Запустите проект: `python bot.py`
